题目分析：
    一.审题
        题干：
            要求用两种方式处理四则运算问题，采用哪种方式由输入的字符决定
            因此，可以直接定义两个函数，一种采用方式0，一种采用方式1来进行计算
            在主函数中控制是采用方式0还是方式1还是两种都进行
        输入格式：输入包含了一个整数 一个浮点数 一个浮点数 一个空格 一个符号 一个回车
            因此，在读取那个符号的时候，需要先排除空格的影响(%*c)
            特别注意：用double类型才能满足精度要求，用float是不能通过提交的
        输出格式：
            相邻两组样例的结果间空一行：在一次完全执行完毕后，需要多输出一个\n来空一行
            注意输出中哪些地方有空格而哪些地方没有空格，尤其是符号的两侧需特别注意有无空格
            注意要用英文符号，不能用中文的各种符号进行输出
    二.程序分析：
        定义两个函数，way0和way1，返回值都是void类型，接受参数为两个计算数num1,num2,和符号syn
        一个函数中采用if-else语句，另一个中采用switch语句来完成任务，
        并都将计算结果储存在本函数内部定义的变量res中，用于输出
        在main函数中，先获取输入，用if语句判断采用哪种方式，再根据不同的方式，
        调用way0或是way1函数，题目要求方式为2时，两种都用，也就是按顺序各调用一次
        输出可以直接在两个way函数中完成，在每个中输出自己对应的输出格式即可
    三.代码片段参考（看了前面的才能看懂）：
        输入示例：
            while (scanf("%d %lf %lf%*c%c", &type, &num1, &num2, &syn)!=EOF)
        函数定义：
            void way0(...){...}
            void way1(...){...}
        计算示例：
            switch (syn) {
                case '+':
                    res = num1 + num2;
                    break;
                case '-':
                    res = num1 - num2;
                    break;
                case '*':
                    res = num1*num2;
                    break;
                case '/':
                    res = (1.0*num1) / num2;
                    break;
                default:
                    break;
            }
        函数调用示例：
            else if (type == 2) {
			    way0(num1, num2, syn);
			    way1(num1, num2, syn);
			    printf("\n");
		    }
        输出示例：
            printf("After switch processing,the result is : %.2f\n", res);
