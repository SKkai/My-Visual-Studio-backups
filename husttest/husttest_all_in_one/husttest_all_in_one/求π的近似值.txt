题目分析：
    一.审题
        题干：
            1.本题给出了π的一个近似值计算公式，必须完全使用这个公式计算，
        而不可根据其他更优秀的计算方式求更精确的值，也不能直接输出π的已知值。
            2.直到最后一项的绝对值小于1e-5的在本题中的意思是（其他题可能并非如此），
        当加到一个绝对值小于1e-5的项时，加上它，并结束计算，输出结果。
        输入格式：无输入
        输出格式：
            保留九位小数：用%.9f进行输出
            输出占一行：仅在输出的结果后面加一个\n
    二.程序分析：
        首先考虑用循环进行连加
        先处理所要加的数的绝对值：用浮点数的除法表示，即用1.0去除以一个每次加2的变量a

        相邻项符号相反有多种方式实现，可以使用一个每次加一的变量flag对二取余来区分奇偶项，
        也可以用(-1)^n来实现

        每次运算的时候都把所要加的数的绝对值用上述方法处理，并加在结果变量pai上
        记得要先初始化pai的值为0

        使用do while语句或while（1） break语句可以更方便的实现对于结束条件的判断
        因为最后一项是要加进去的，如果只是用while语句可能导致少加了最后一项

    三.易错代码片段参考（看了前面的才能看懂）：
        1.奇偶判断：
            if (flag % 2) {
                pai += 1.0 / a;
            }
            else {
                pai -= 1.0 / a;
            }
            flag++;
		2.结束条件判断：
            if ((1.0 / a) < 1e-5)
            {
                break;
            }
        3.输出：
            printf("%0.9f\n", pai);
